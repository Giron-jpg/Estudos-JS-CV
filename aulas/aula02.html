Iniciando estudos com DOM

DOM significa Document Object Model, que é um Modelo de Objetos para Documentos, que nada mais é que o conjunto de objetos dentro de um HTML.

Para entender melhor o funcionamento do DOM estudaremos a Árvore DOM. A imagem abaixo mostra a Árvore DOM deste documento até aqui.

Esses objetos permitem a navegaçáo para a manipulação dos scripts do site. Caso haja necessidade em pegar um elemento específico usamos o metodo getElement(s) que pode selecionar: um tag (ByTagName), por id (ById), por name (ByName), por class (ByClassName) ou por seletor. Se o uso do get for atribuido há elementos com vários objetos é necessário informar entre colchetes qual elemento pretende-se usar.

No caso do seletor (selector) informado no parágrafo anterior o sistema é um pouco diferente. O uso do método utiliza o método querySelector(), onde o parâmetro recebido informa a tag e sua classe (class) ou identificação (id). Ex.: querySelector('div#img')
